API Venda de Produtos - Desafio técnico PRODAP - Sênior.

API foi desenvolvida com tecnologia Asp .Net Core 2.2
As informações estão sendo gravadas em memória, não sendo necssário a criação de banco de dados físico para realização de testes.
Para facilitar os testes, foi configurado o Swagger UI para realização das requisições.

Foi criado as seguinte funcionalidades:

- Registrar Produto
- Listar Produtos
- Registrar funcionários
- Listar funcionários
- Registrar saída de vendas
- Registrar vendas
- REgistrar Estoque
- Foi implementado regras de validação de estoque, não permitindo a saída de uma quantidade produtos maior que o saldo em estoque.
